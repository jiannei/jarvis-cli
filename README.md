# jarvis-cli

GitHub 访问加速工具 - 提供 DNS 优化、镜像加速、代理配置等功能

## 安装

```bash
cargo install jarvis-cli
```

## 使用

```bash
jarvis
```

## 功能

- **直连模式** - 优化 DNS/hosts，直接访问 GitHub
- **镜像模式** - 配置 Git 镜像加速
- **代理模式** - 配置 Git 代理
- **SSH 模式** - 配置 SSH 密钥
- **连接测试** - 测试 GitHub 连接状态
- **自动测速** - 自动选择最快的访问方式
